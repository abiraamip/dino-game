# dino-game
A Chrome-inspired dino runner game built using an Arduino Uno and an SSD1306 OLED display

🔌 Hardware Components: Arduino Uno, 0.96" SSD1306 OLED Screen, push button, and breadboard

🛠️ Pin Mapping: 
OLED (I2C)
OLED Pin	Arduino Uno
VCC	5V
GND	GND
SDA	A4
SCL	A5

Push Button
Button Pin	Arduino Uno
One side	D2
Other side	GND

📚 Required Libraries: Links to Adafruit_SSD1306 and Adafruit_GFX.
